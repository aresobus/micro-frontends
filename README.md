# micro-frontends

## HI There!

### To run this project on your local:

```
1. npm install
2. yarn start
3. navigate to http://localhost:8080/
```

#### App build with:
*Single-SPA (a javascript framework for front-end microservices) 
*Angular
*React

Please find detailed documentation here:
https://single-spa.js.org/


Have fun!
